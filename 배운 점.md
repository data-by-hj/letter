## views
### request 객체
request = 사용자의 요청 정보를 담은 객체
#### 주요속성
```
request.method       # 요청 방식
request.user         # 현재 사용자
request.POST         # POST (전송) 데이터
request.GET          # GET (보기)데이터
request.FILES        # 업로드 파일
request.path         # URL 경로
request.META         # 메타 정보
request.COOKIES      # 쿠키
request.session      # 세션
```
주로 사용하는 함수 :
데이터 전송 - request.POST.get('username')
사용자 확인 - authenticate(request, 로그인 속성=값, 로그인 속성=값)
로그인 처리 - login(request, user)
로그아웃 처리 - logout(request)

#### redirect vs render
redirect: URL로 이동 (주소창 바뀜) 데이터 처리 후 사용
render: HTML 보여주기 (주소창 안 바뀜) 폼 보여줄 때 사용

#### 객체 생성, 변경, 조회
인스턴스 생성 - 테이블.objects.create(속성=값, 속성2=값2)
인스턴스의 특정 속성 값 변경 - 인스턴스.속성 = 값
조회 - 테이블.objects.filter(속성=값)
비밀번호 변경 - request.user.set_password(new_password)
인스턴스 변경 후 저장 - 인스턴스.save()
